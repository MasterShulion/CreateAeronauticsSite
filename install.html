Quick setup

How to install the modpack

NeoForge 1.21.1.230 for Minecraft 1.21.1 • 4 GB RAM minimum

1. Requirements

  • Java 17 or newer
  • NeoForge 1.21.1.230 (for Minecraft 1.21.1)
  • At least 4 GB RAM for the game

2. Download the modpack

Click Open Modpack on the homepage or use the Google Drive link. After download, check SHA256 checksum.

3. Install NeoForge

  1. Create a new Minecraft 1.21.1 profile in launcher.
  2. Install and select NeoForge 1.21.1.230.
  3. Set at least 4 GB RAM.

4. Import modpack

If your launcher supports import, use it. If not, copy mods and config into .minecraft.

5. Common issues & fixes

  • Crash on start: Check launcher logs and confirm NeoForge 1.21.1.230 is selected.
  • Missing mods: Re-download modpack and re-import it fully.
  • Black screen: Update GPU drivers and increase RAM.

Быстрая установка

Как установить модпак

NeoForge 1.21.1.230 для Minecraft 1.21.1 • минимум 4 ГБ ОЗУ

1. Требования

  • Java 17 или новее
  • NeoForge 1.21.1.230 (для Minecraft 1.21.1)
  • Минимум 4 ГБ ОЗУ для игры

2. Скачай модпак

Нажми Открыть модпак на главной или скачай по ссылке Google Drive. После скачивания проверь SHA256.

3. Установи NeoForge

  1. Создай новую установку Minecraft 1.21.1 в лаунчере.
  2. Установи и выбери NeoForge 1.21.1.230.
  3. Выдели минимум 4 ГБ ОЗУ.

4. Импортируй модпак

Если лаунчер умеет импорт, просто импортируй модпак. Если нет, скопируй папки mods и config в .minecraft.

5. Частые проблемы и их решения

  • Краш при запуске: Проверь логи и убедись, что выбрана версия NeoForge 1.21.1.230.
  • Не хватает модов: Скачай модпак заново и импортируй полностью.
  • Чёрный экран: Обнови драйвер видеокарты и дай больше ОЗУ.
